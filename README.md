# leadlens-ai

### How to Load the Plugin into Google Chrome
Follow these steps to install the plugin in Google Chrome:

### Download the ZIP File
First, download the ZIP file containing the plugin.
<img width="1440" alt="image" src="https://github.com/user-attachments/assets/bdd8e7ee-087c-4849-be14-3132c5ceeadb">

### Extract the ZIP File
Extract the contents of the ZIP file to get a folder named build.
<img width="907" alt="image" src="https://github.com/user-attachments/assets/c02c054c-e6ee-4d9b-9e90-1e5ed0d54524">
<img width="913" alt="image" src="https://github.com/user-attachments/assets/bcd7db5d-58fb-4c75-92fa-afde85aa19c4">

### Open Google Chrome
Launch Google Chrome and click on the puzzle piece icon (extensions) in the top right corner of the browser.
<img width="1440" alt="image" src="https://github.com/user-attachments/assets/371e70cf-9128-4624-9265-610dd0cc3837">


### Enable Developer Mode
Scroll down and enable "Developer mode" by toggling the switch in the top right corner of the Extensions page.
<img width="1440" alt="image" src="https://github.com/user-attachments/assets/1b34eee8-7bb6-44d2-89a6-75161c9fbb50">

### Load Unpacked Extension
Click on "Load unpacked" and select the build folder you extracted earlier.
<img width="647" alt="image" src="https://github.com/user-attachments/assets/c00fd0fe-07e7-45ec-bac3-5be9dd9bb5b0">
<img width="796" alt="image" src="https://github.com/user-attachments/assets/fe0e2af6-a910-4f2d-b235-5c267289ee73">

### Pin the Plugin
After loading the extension, click on the puzzle piece (extensions) again, find the plugin, and click the pin icon to pin it to your toolbar.
<img width="486" alt="image" src="https://github.com/user-attachments/assets/f9625575-2721-4451-a292-7ac96d20b8a6">

### Register and Login
Go to the website, register for an account, and use the same credentials to log in to the plugin.
